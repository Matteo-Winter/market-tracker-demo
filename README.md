# Market Tracker Demo

This is a public demo version of an Amazon Market Tracker project.

The goal of the project is to look beyond single-product research and track broader market movements over time. Instead of only checking individual products, the tool is designed to analyze categories, long-term changes, competitive movement, and emerging or fast-growing products.

## Important Note

This repository is **not the latest internal version** of the project.

It is a cleaned public demo version that only shows parts of the project I was able to publish. Some features, data, internal logic, and newer improvements were removed or are not included in this repository.

## What the project is about

Most Amazon research tools focus on product snapshots. This project explores a different approach: following whole markets over time.

The idea is to make it easier to understand:

- how categories change
- which products are starting to grow
- where competition is moving
- what trends appear over longer periods
- where new opportunities might be forming

## Project Status

This is a demo/prototype version.

It is mainly meant to show the concept, structure, and direction of the project. It is not intended to be a complete production-ready release.
